Application de gestion des salles: 

Noureddine Kheloufi 300151753


description:
Cette application permet de gérer facilement des salles à travers une interface graphique simple. On peut ajouter, modifier, supprimer ou rechercher des salles, avec toutes les données enregistrées dans une base MySQL. Le projet est organisé de manière claire pour bien séparer les différentes parties du programme.
